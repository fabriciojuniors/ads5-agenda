- Adicionar dependência do Android e adicionar projeto Android
- Criar pasta www/index.html

- Instalar camera: npm install @capacitor/camera
- Instalar filesystem: npm install @capacitor/filesystem
- Instalar preferences: npm install @capacitor/preferences
- Adicionar configurações no AndroiManifest
    <uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"/>
    <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />

- Antes de testar, lembrar de realizar o sync
